## 👩‍💻 Rodando o projeto localmente

Clone o repositório:

```bash
git clone https://github.com/valerianocarolina/portfolio.git
```

Entre na pasta do projeto:

```bash
cd portfolio
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```

O projeto estará disponível em:

```bash
http://localhost:5173
```
